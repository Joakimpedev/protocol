## Voice: The Stoic Warrior

- Calm, direct, assumes intelligence
- Complete sentences, natural grammar
- No hype, no apologies, no forced motivation
- States what is true and what to do
- Respects the user's time and mind

Think Marcus Aurelius. Wise, deliberate, clear. Says what needs to be said, nothing more.

## Visual: Terminal-Evolved

| Element | Direction |
|---------|-----------|
| Headings/Labels | Monospace font |
| Body text | Clean sans-serif |
| Background | Dark, not pure black (~`#0a0a0a`) |
| Text | Off-white, not harsh (~`#e5e5e5`) |
| Accent color | Minimal or none |
| Corners | Sharp, minimal rounding |
| Density | Not afraid of text, readable but dense |

The terminal spirit (structure, monospace, density) without looking like a code editor.

## Copy Rules

**Do:**
- Write complete, natural sentences
- Be direct about what works and why
- State facts, let the user decide
- If something is hard, say it's hard
- If they failed, say they failed - then say to begin again

**Don't:**
- Use emojis
- Use exclamation marks
- Say "great job", "you got this", "amazing"
- Use "bro" or casual slang
- Overcomplicate with medical jargon
- Soften bad news
- Pump them up artificially
